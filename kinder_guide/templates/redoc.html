AUTH

Регистрация пользователей и выдача токенов

Регистрация нового пользователя

Получить код подтверждения на переданный email. Права доступа: Доступно без токена Поля email и phone_number должны быть уникальными. Пароль должен быть не менее 8 символов, содержать заглавные и строчные латинские буквы и цифры. Поля password и password2 должны совпадать.

Request Body schema: application/json
name
required
string <= 150 characters ^[\p{L}\w.@+-]+\z
surname
required
string <= 150 characters ^[\p{L}\w.@+-]+\z
phone_number
required
string <= 15 characters ^\+\d{1,3}[\d()-]{6,14}\d$
email
required
string <email> <= 254 characters
password
required
string >= 8 characters (?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])
password2
required
string >= 8 characters (?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])

Responses

/api/v1/auth/signup/
